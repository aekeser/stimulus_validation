# Multimodal Sentence-Picture Validation Study

This repository contains materials for a PCIbex/PennController-based multimodal sentence-picture validation experiment developed at the University of Georgia.

## Study Purpose

The purpose of this study is to validate auditory and visual experimental materials for a future sentence-processing and eye-tracking research project.

The validation study evaluates sentence-picture mapping validity, foil discriminability, auditory intelligibility, auditory naturalness, visual recognizability, and overall multimodal item quality.

This is a methodological stimulus-validation study and not a grammaticality judgment task.

## Experimental Framework

The experiment was implemented using PCIbex, PennController, and GitHub Pages external media hosting.

## Trial Structure

Each validation trial includes visual display onset, auditory sentence playback, participant picture selection, and post-selection ratings.

Participants complete ratings for sentence-picture match, audio clarity, visual clarity, and overall item quality.

## Repository Contents

This repository contains stimulus image files, auditory stimulus files, the PCIbex/PennController experiment script, and presentation list CSV files.

Raw participant data are not stored in this repository.
